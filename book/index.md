# PanDownload

#### 软件介绍

本软件为 PanDownload 的众多修改版中的一个，制作本软件主要是因为原版 PanDownload 由于某个众所周知的原因，已经没办法正常使用了。

鉴于找不到比 PanDownload 更优秀的百度网盘多账号管理软件，所以对 PanDownload 进行了一些基础功能上的修复。

随着最近一些其他修改版的维护者退坑。故而现在放出这一自用版本发布给有需要的人使用。希望大家喜欢。
